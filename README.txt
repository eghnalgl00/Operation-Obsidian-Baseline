# Operation Obsidian — Templater Scripts Pack (Fix)
Drop the folders into your **vault root**.

## Then in Obsidian → Settings
Templater → *Script files folder location* = `.obsidian/scripts`  
Templater → Startup templates:
- `Templates/Startup_Auto_Daily_Log.md`
- `Templates/Startup_Auto_XP.md`

Hotkeys (optional): bind the two template files to keys.
- Daily Log: `Templates/Startup_Auto_Daily_Log.md`
- Update XP: `Templates/Startup_Auto_XP.md`

After any change: **Templater → Reload User Scripts**.
